# 도형 합성 퍼즐 게임 기획서

## 1. 핵심 메커니즘

### 게임 방식

- **합성 시스템**: 원 + 원 → 삼각형 → 사각형 → 오각형 → 육각형
- **목표**: 육각형 2개를 합쳐 소멸시키며 최대한 오래 생존 또는 높은 점수 달성
- **게임오버**: 빈 칸이 없고 더 이상 합칠 수 없을 때

### 판 크기 확장

- **3x3**: 게임 시작
- **4x4**: 누적 500점 또는 2분 생존
- **5x5**: 누적 2000점 또는 5분 생존
- 확장 시 기존 배치 유지, 가장자리에 빈 칸 추가

## 2. 레어도 시스템

| 등급 | 색상 | 확률 | 효과 |
| --- | --- | --- | --- |
| 커먼 | 흰색 | 70% | 기본 아이템 |
| 레어 | 파란색 | 20% | 합칠 때 1.5배 점수 |
| 에픽 | 보라색 | 8% | 2배 점수 + 주변 같은 모양 1개 자동 합치기 |
| 레전더리 | 금색 | 2% | 3배 점수 + 원하는 위치로 이동 가능 |

## 3. 점수 시스템

**점수 계산식**

`(아이템 단계 × 10) × 레어도 배율 × 콤보 배율`

**아이템 단계 점수**

- 원: 1단계 (10점)
- 삼각형: 2단계 (20점)
- 사각형: 3단계 (30점)
- 오각형: 4단계 (40점)
- 육각형: 5단계 (50점)

**콤보 시스템**

- 3초 내 연속 합치기 시 콤보 증가
- 1콤보: x1.2 / 2콤보: x1.5 / 3콤보+: x2.0
- 3초 경과 시 콤보 초기화

## 4. 게임 진행

### 소환 메커니즘

- 빈 칸에 랜덤 위치로 소환
- 현재 판에서 가장 높은 단계보다 낮은 아이템만 소환
- 레어도는 확률에 따라 랜덤 부여

### 소환 속도 (난이도 조절)

- 초반 (0-2분): 3초마다
- 중반 (2-5분): 2초마다
- 후반 (5분+): 1초마다

## 5. 특수 아이템/스킬

게임당 사용 횟수 제한

| 스킬 | 효과 | 사용 횟수 |
| --- | --- | --- |
| 리롤 | 다음 소환 아이템 변경 | 3회 |
| 폭탄 | 선택한 아이템 제거 | 1회 |
| 시간 정지 | 5초간 소환 중지 | 1회 |

## 6. 추가 콘텐츠

### 미션/도전 과제

- "30초 안에 육각형 만들기"
- "한 판에서 10콤보 달성"
- "레어 아이템만으로 사각형 만들기"
- "스킬 사용 없이 1000점 달성"

### 시각적 피드백

- 같은 모양 인접 시 테두리 발광
- 합칠 수 있는 조합 힌트 표시 (옵션)
- 레어도별 파티클 이펙트
- 콤보 달성 시 화면 효과

## 7. 게임 모드

### 클래식 모드

- 무한 생존, 최고 점수 도전

### 타임 어택

- 3분 내 최대 점수

### 미션 모드

- 특정 조건 달성 (육각형 N개 만들기 등)
